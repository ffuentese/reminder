# reminder

This app aims to create a extremely basic password keeper with Python and SQLite.
Don't use it for real but you can test it.

Python 2.7 must be installed on the system.
You can execute it directly from Explorer double clicking on gestor.py.

It uses the TKinter library to display a GUI.
http://effbot.org/tkinterbook/
